About Android


Welcome to Android! It is a great mobile operating system and an enjoyable one to developed for. You will discover putting together a decent looking Android app is not hard to do, but getting it to Google Play is. There are a lot of little things to worry about.



About Android:

  • Founded in October 2003 by Andy Rubin, Rich Miner, Nick Sears, and Chris White
  • Its open source
  • Acquired by Google in 2005
  • 82% of the smart phone users have Android Phones

http://www.idc.com/prodserv/smartphone-os-market-share.jsp


Getting started

Lets get started! Google is currently phasing out the Android Plugin for Eclipse so we are going to use Android Studio.
Here is the current stable version of Android Studio (1.5):
https://developer.android.com/sdk/index.html

Or you can use the preview version (2.0):
http://tools.android.com/download/studio/builds/2-0-preview

If you run into any issues when installing Android Studio, please tell me by filing a bug. It has been a while since I've installed it.
